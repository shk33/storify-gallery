# Instagram Gallery: Storify API example written in Node.js

This is the first example app in the book Appres Pro Express.js 
An example app using express.js

## Dependencies

* express: 4.8.1
* superagent: 0.18.2
* consolidate: 0.10.0
* handlebars: 2.0.0

## Running the app
```
node index.js
```
And visit http://localhost:3001
